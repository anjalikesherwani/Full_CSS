CSS complete intro

What is css

css stands for (cascading style sheet).

describes how html elements are to be displayed on screen.

CSS covers multiple layout at once.

type of css

Internal css:- An internal stylesheet holds CSS rules for the page in the head section of the HTML file.

External css:- An external stylesheet is a file containing only CSS syntax.

Inline css:- Inline CSS allows you to apply a unique style to one HTML element at a time.

How css works

css selector

In css, selectors are patterns used to select the element(s) you want to style.

Two main selectors.

  • .class (eg .intro):- Selects all elements with class = "intro"
  • #id (eg #firsrname):- Selects the element with id="firstname"



Working with Internal css

CSS properties

  • width
    • gives widtn to the html element
    • in px or %
  • height
    • gives height to the html element
    • in px or %
  • background-color
    • give background color to the html element
    • color name or color code
  • float
    • give direction to the html element
    • left or right
